Position Property with the value Relative

Position relative with no properties set.

.position-relative {
position: relative;
border: solid RebeccaPurple 3px;
}


A relatively positioned element with no additional properties set will display as normal. It would appear the same as position: static. You need to add extra properties to see any difference. See below.


Position relative with properties set.

.position-relative {
position: relative;
border: solid RebeccaPurple 3px;
}
