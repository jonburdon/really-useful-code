Position Property with the value Relative

Position relative with no properties set.

.position-relative {
position: relative;
border: solid RebeccaPurple 3px;
}


A relatively positioned element with no additional properties set will display as normal. It would appear the same as position: static. You need to add extra properties to see any difference. See below.


Position relative with properties set.

.position-relative {
position: relative;
border: solid RebeccaPurple 3px;
}

.position-relative-topleft {
position: relative;
top: -30px;
left: 100px;
max-width: 400px;
background-color: #fff;
border: solid RebeccaPurple 3px;
}

.position-relative-right {
position: relative;
right: 50px;
text-align: right;
max-width: 400px;
background-color: #fff;
border: solid RebeccaPurple 3px;
}

DIV with class position-relative-topleft

This div has top: -30px; and left: 100px; properties set. The width and background-colour have also been set.

top: -30px; moves it up by 30px from it's normal position ie inline, block or inline block.

left: 100px; moves it 100px away from the left. ie it moves to the right.

Note that the gap below is caused by the fact that this div has been moved up.
