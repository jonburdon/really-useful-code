Margins and Widths

Width 500px

.width500 { width: 500px; }


Div with width 500px.

Width 500px margin auto

.width500-marginauto { width: 500px; margin: auto; }


Div with width 500px and margin set to auto. Margin is split between left and right.

Max Width 800px

.maxwidth800-marginauto { max-width: 800px; margin: auto; }


Div with max width 800px and margin set to auto. When the browser is small, the width will reduce below 800px. Useful!

Width 70%

.width70percent-marginauto { width: 70%; margin: auto; }


The div will always by 70% of the viewport width.
