Box Sizing

Standard 500px box

.width500 {
width: 500px;
border: solid RebeccaPurple 3px;
}


Div with width 500px. Well, 506px if you include the border.

Width 500px Box with Styling

.width500 {
width: 500px;
}
.styledbox {
margin: 30px;
padding: 60px;
border: solid RebeccaPurple 15px;
}


Div with width 500px and margin, padding, border width added. The overall box width is larger than 500px because actual heigh of element is height+padding+border.

Width 500px with Styling and Box Sizing

.width500 {
width: 500px;
}
.styledbox-borderbox {
margin: 30px;
padding: 60px;
border: solid RebeccaPurple 15px;
}


Div with width 500px and box-sizing: border-box applied. The div is now 500px wide despite having the same padding, margin and border as above.
