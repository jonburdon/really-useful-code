The Display Property - Part 1 - Block and Inline

Block Elements

.purpleborder { border: solid RebeccaPurple 3px; }


Div with a border and no other properties.
Another div with the same simple border. Because these are block level elements, they will take up the whole width of the screen. Paragraphs and headings are common block level elements.

Inline Elements

.purpleborder { border: solid RebeccaPurple 3px; }


A span with a border and no other properties. A span is a typical example of an inline html element. Another span with the same simple border.Because these are inline elements, they will only take up the space of their box (margin, border, padding and content.) a elements are common inline elements.
