The Display Property - Part 1 - Block and Inline

Block Elements

.purpleborder { border: solid RebeccaPurple 3px; }


Div with a border and no other properties.
Another div with the same simple border. Because these are block level elements, they will take up the whole width of the screen. Paragraphs and headings are common block level elements.

Inline Elements

.purpleborder { border: solid RebeccaPurple 3px; }


A span with a border and no other properties. A span is a typical example of an inline html element. Another span with the same simple border.Because these are inline elements, they will only take up the space of their box (margin, border, padding and content.) a elements are common inline elements.

Inline Elements

.purpleborder { border: solid RebeccaPurple 3px; }


You can use the display css property with the inline-block value
An element with the inline-block value selected does not add a line break after the element.
Importantly, an element displaying as inlne-block can also have the width and height set, which inline elements cannot.

A span displayed as inline

.inline { display: inline; border: solid RebeccaPurple 3px; width=100px; height=100px; }


This is an identical paragraph of text. Within the paragraph is a span element. This span is displaying with the above code.The same element is being displayed three times, the only different being whether it is displayed a inline, block, or inline-block. As an inline element, width and height do not display.
